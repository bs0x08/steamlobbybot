SteamLobbyBot ToS

1. Usage
This bot is provided free of charge to allow Discord users to check who is playing Steam games among registered members of their server.

2. Data Collected
The bot stores only the following information:
- Discord user ID
- Discord server ID
- SteamID64 (linked by the user)
