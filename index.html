SteamLobbyBot Privacy Policy

1. Information Collected
The bot stores the following:
- Discord user ID
- Discord server ID
- SteamID64 (as provided by the user)

2. Purpose
This data is used only to check which registered users are currently playing Steam games.

3. Data Sharing
The bot does not share stored data with third parties.

4. Data Retention
Data is stored until the user unregisters with /unregisterid or the server removes the bot.

5. Contact
If you have any concerns about privacy, contact the developer of SteamLobbyBot.
